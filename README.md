# AI Investor Copilot 

## Setup

### Backend
cd backend
pip install -r requirements.txt
pyhton main.py

### Frontend
Open frontend/index.html

## Features
- Multi-agent AI system
- Real stock data (NSE via Yahoo Finance)
- Moving Average + RSI strategy
- Decision engine (BUY / WATCH / AVOID)
- Top opportunity scanner
- Clean UI

## Agents
- Data Agent
- Signal Agent (MA + RSI)
- Decision Agent
- Explanation Agent

## Demo Flow
1. Load Top Opportunities
2. Analyze stock
3. View signals + decision

## Future Scope
- LLM explanations
